\# Eastern India's Hidden Gems



A multi-page travel exploration website showcasing lesser-known

natural and sacred destinations across Eastern India.



\## Overview

This project curates 126 offbeat locations including waterfalls,

sacred landscapes, dams, forests, and tribal heritage sites.

The focus is on clean UI, structured navigation, and data-driven exploration.



\## Pages Included

\- Home (Dynamic Explorer)

\- Destinations

\- Experiences

\- Travel Guide

\- Blog

\- About

\- Contact



\## Key Features

\- Fully interlinked multi-page website

\- Dark-themed responsive UI

\- Dynamic filtering and modal views

\- Consistent design system and navigation

\- Structured dataset integration



\## Tech Stack

\- HTML5

\- CSS3

\- Vanilla JavaScript



\## Purpose

Built as a personal project to practice front-end development,

information architecture, and UI consistency.
